# 📚 Booknest

A simple, clean book-tracking web app where you can add books with their author and a short description, then remove them whenever you like — all in the browser, no backend required.

## Features

- Add books with a title, author name, and short description
- Popup form with overlay for adding new entries
- Delete any book card with a single click
- Responsive card-based layout
- Clean UI styled with the Poppins font

## Demo

Open `index.html` in your browser — no build steps or installation required.

## Tech Stack

- **HTML5** for structure
- **CSS3** for styling (flexbox-based card layout, custom popup/overlay)
- **Vanilla JavaScript** (DOM manipulation, event listeners) — no frameworks or libraries

## Project Structure

```
Booknest/
├── index.html      # Markup and structure
├── style.css       # Styling
├── script.js       # App logic (add/delete books, popup toggle)
└── README.md
```

## How It Works

1. Click the **+** button to open the "Add Book" popup.
2. Fill in the book name, author, and a short description.
3. Click **ADD** to create a new book card, or **CANCEL** to close the popup without saving.
4. Click **Delete** on any book card to remove it.
